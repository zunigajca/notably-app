# Notably — Your Intelligent Creative Workspace

Notably is a modern, full-stack MERN dashboard designed to streamline note-taking. It features an interactive UI, rapid search filtering, and smart tag suggestions to keep your thoughts perfectly organized with minimal clutter.

## 🚀 Live Demo
You can view the live frontend application here: [notably-app-six.vercel.app](https://notably-app-six.vercel.app/)

## 🛠️ Tech Stack
- **Frontend:** React, Vite, Tailwind CSS, Axios
- **Backend:** Node.js, Express.js, Mongoose
- **Database:** MongoDB Atlas
- **Hosting:** Vercel (Frontend) & Render (Backend)

## ✨ Features
- **Dynamic Search Filtering:** Instantly filter your workspace by note title or keyword directly from the search bar.
- **Smart Tag Suggestions:** Automatically extracts your top 6 most frequently used tags to provide quick-add pills under the input field, keeping your categorization tidy and consistent.
- **Interactive Hashtags:** Click on any tag pinned to a note card to instantly filter your workspace to show only notes carrying that specific tag.
- **Full CRUD Functionality:** Seamlessly create, read, edit, and delete notes in real time.
